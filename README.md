# PROJECT EGO — Путь от бариста к AI-создателю

Личная RPG-система управления жизнью, карьерой, финансами, контентом, здоровьем и семьёй.
Работает полностью локально, без сервера, без регистрации, без платных API.

---

## 1. Как запустить (самый простой способ)

1. Создай на компьютере папку, например `PROJECT-EGO`.
2. Положи в неё три файла из этого набора:
   - `index.html`
   - `styles.css`
   - `app.js`
3. Дважды кликни по `index.html` — приложение откроется в браузере.

Всё. Никаких установок, команд и терминалов.

> Если файлы создаёшь вручную: открой «Блокнот» (лучше — бесплатный VS Code),
> вставь код соответствующего файла, сохрани с точным именем (`index.html`, `styles.css`, `app.js`)
> в одну и ту же папку. Кодировка — UTF-8 (в блокноте выбирается при «Сохранить как»).

## 2. Первый запуск

При первом открытии появится мастер настройки из 8 шагов.
Данные уже предзаполнены из твоего плана (долги, цели, график) — просто проверь и поправь.
После мастера появятся демо-квесты для примера. Удалить их можно одной кнопкой:
**Настройки → «Очистить демо-данные»**.

## 3. Где хранятся данные

В `localStorage` твоего браузера — на твоём компьютере, никуда не отправляются.
Данные переживают перезапуск браузера и компьютера.

⚠ **Важно:** данные привязаны к браузеру. Если очистить «данные сайтов» в настройках
браузера — прогресс удалится. Поэтому регулярно делай резервную копию (см. ниже).
Приложение само напомнит, если копии не было больше двух недель.

## 4. Резервная копия

1. Открой **Настройки → «Экспорт JSON»**.
2. Файл `project-ego-backup-ДАТА.json` сохранится в загрузки.
3. Храни его в облаке (Яндекс.Диск, Google Drive) или на флешке.

Рекомендация: делай экспорт раз в неделю, после недельного обзора.

## 5. Перенос на другой компьютер или телефон

1. На старом устройстве: **Настройки → Экспорт JSON**.
2. Скопируй папку со всеми файлами приложения + JSON-файл на новое устройство.
3. Открой `index.html`, быстро пройди мастер (любые значения).
4. **Настройки → Импорт JSON** → выбери свой файл. Всё восстановится.

## 6. iPhone: как пользоваться с телефона

Приложение — это PWA: его можно установить на iPhone как обычное приложение.

1. Сначала опубликуй его бесплатно (следующий раздел) — iOS требует https.
2. Открой адрес в **Safari** на iPhone.
3. Нажми «Поделиться» (квадрат со стрелкой) → **«На экран „Домой"»** → «Добавить».
4. На домашнем экране появится иконка **EGO**. Открывается на весь экран,
   без адресной строки, работает **без интернета** (файлы кэшируются автоматически).

Важно про данные:
- На телефоне и компьютере — независимые копии данных. Перенос прогресса:
  «Настройки → Экспорт JSON» на одном устройстве → «Импорт JSON» на другом
  (JSON-файл можно переслать себе в Telegram «Избранное»).
- Если пользоваться просто во вкладке Safari (без установки на экран «Домой»)
  и не открывать сайт ~7 дней, iOS может стереть локальные данные.
  Установка на экран «Домой» защищает от этого, но еженедельный экспорт — всё равно золотое правило.
- Удобный сценарий: телефон — основное устройство (отмечать квесты в течение дня),
  компьютер — для разработки проектов; раз в неделю переносить бэкап.

## 7. Как опубликовать бесплатно (нужно для телефона)

Самый простой путь — GitHub Pages:

1. Зарегистрируйся на github.com (бесплатно).
2. Создай репозиторий, например `project-ego` (можно приватный → тогда используй Vercel/Netlify, для Pages нужен публичный).
3. Загрузи все файлы из папки через кнопку «Add file → Upload files».
4. В настройках репозитория: **Settings → Pages → Source: main branch → Save**.
5. Через минуту приложение будет по адресу `https://твой-ник.github.io/project-ego/`.

Альтернативы: Netlify Drop (просто перетащить папку на netlify.com/drop), Vercel.
Учти: на каждом устройстве будет своя копия данных (localStorage локален) — переноси прогресс через экспорт/импорт.

Кстати, публикация этого приложения — готовый первый пункт портфолио и тема для первого ролика.

## 8. Ключевые механики (краткая шпаргалка)

- **Режим дня** — выбирай честно: рабочий / выходной / низкая энергия / восстановление. Нагрузка подстроится.
- **Минимальный день** — 10 минут карьеры + здоровье + семья + отметка энергии. Держит серию в тяжёлые дни.
- **Возвращение героя** — пропустил день(и)? Серия не обнуляется: её спасают жетоны 🛡, а без них сохраняется 60%. Просто вернись.
- **Стабильность за 28 дней** — главная метрика. Она важнее серии.
- **Лимиты** — не больше 2 активных проектов, 5 сезонных целей, 3 главных задач в день. Это защита от тебя самого.
- **Gold** — внутренняя валюта наград. Сон, еда и семья бесплатны всегда.
- **Босс-квесты** — измеримые дела бьют по HP босса сезона.
- **Недельный обзор** (воскресенье) — 10 минут, даёт жетон защиты и собирает неделю в картину.

## 9. Финансовый дисклеймер

Все финансовые расчёты в приложении — ориентировочные. Они не заменяют
банковскую выписку, график платежей банка или консультацию специалиста.
Вноси реальные остатки по долгам — тогда прогнозы будут ближе к правде.

## 10. Структура файлов

```
PROJECT-EGO/
├── index.html            — каркас приложения
├── styles.css            — стили, 3 темы, safe-area для iPhone
├── app.js                — вся логика (~3000 строк, с комментариями по разделам)
├── manifest.webmanifest  — PWA-манифест (установка на телефон)
├── sw.js                 — service worker (офлайн-режим, работает по https)
├── icon-180.png          — иконка для iPhone (apple-touch-icon)
├── icon-192.png, icon-512.png — иконки PWA
└── README.md             — этот файл
```

Код структурирован по нумерованным разделам (см. оглавление в начале `app.js`) —
это удобная база для доработок через Cursor или другой AI-редактор.
Расширение схемы данных: увеличивай `version` в состоянии и дополняй `migrate()`.
